# Lakota Language Apps — Market Research
*Traditionally Different Technology | Research Date: March 2026*

---

## Overview

All three apps are products of the **Lakota Language Consortium (LLC)** and its sister organization **The Language Conservancy** — a nonprofit duopoly that dominates the digital Lakota language space. They form a loosely integrated ecosystem: reference layer (NLD), input layer (LKT Keyboard), and learning layer (Owóksape).

---

## App 1: New Lakota Dictionary (NLD) Mobile

**App Store:** https://apps.apple.com/us/app/new-lakota-dictionary-mobile/id959701704
**Developer:** Lakota Language Consortium, Inc.
**Price:** $9.95 (free lifetime updates)
**Platforms:** iOS, Android, Desktop (Pro — $29.95), Physical Book ($49.95)
**Bundle ID:** com.lingea.handylex.lkenh1
**Latest Version:** 4.2 (released Feb 2024)
**Min iOS:** 12.0

### Features
- 26,000 entries, 40,000 example sentences
- Male and female native speaker audio recordings
- Works fully offline — no data connection needed
- Standard Lakota Orthography (SLO) throughout
- Built-in lemmatizer: resolves conjugated/inflected verb forms to root
- Inflection charts showing complete paradigms
- Bidirectional search: English → Lakota and Lakota → English
- Cross-referenced hyperlinked entries
- Favorites lists and search history
- Built-in SLO keyboard for special characters
- Copy/paste by word or line
- User feedback/comment system per entry
- Over 400 native speakers involved in development and review

### Strengths
- The **lemmatizer** is the flagship differentiator — handles complex Lakota verb affixation automatically, something a print dictionary cannot do
- Deep community trust: co-developed with native speakers
- Companion to the physical NLD book (the authoritative reference)
- Also available as desktop software — cross-platform ecosystem

### Known Gaps (from reviews)
- Navigation between cross-referenced words is clunky (no back-swipe)
- No sentence construction guidance (users want grammar context, not just entries)
- Update cadence questioned — some users unsure if actively maintained
- Web version was discontinued mid-2024; online platform now offline
- No AI/conversational component
- No pronunciation recording/feedback for learners

### Competitive Position
The authoritative Lakota dictionary. No real competitor at this depth. The physical book + app + desktop software form a moat, but the UX is showing age.

## App 2: LKT Keyboard (Lakota Keyboard — Mobile)

**App Store:** https://apps.apple.com/us/app/lakota-keyboard-mobile/id972130229
**Developer:** Lakota Language Consortium, Inc.
**Price:** Free
**Platforms:** iOS (robust), Android (dated — Samsung only, largely deprecated)
**Min iOS:** 8.1
**Last Update:** Version 2.0 (2015 — updated by Apple for Watch icon only)

### Features
- iOS system keyboard extension (works in any text field system-wide)
- Standard Lakota Orthography character set
- Globe icon to switch between Lakota and English
- Works in email, SMS, social media, notes, any app

### Strengths
- Solves a real problem: typing diacritical Lakota characters system-wide
- Uses same SLO as NLD — consistent orthography across ecosystem
- Companion use with NLD is natural
- Free

### Known Gaps (from reviews)
- **Defaults to all-caps** — requires extra toggling to reach lowercase (most common use case)
- Stress-mark vowels on a secondary screen — extra toggles required
- No long-press diacritic option (modern standard — Gboard and iOS native do this better)
- Android app is essentially abandoned; LLC now directs Android users to enable Lakota via Gboard settings instead
- No updates in ~10 years
- Privacy policy not provided to Apple

### Competitive Position
Utility-only. The app is functionally obsolete on Android (Gboard natively supports Lakota now). On iOS it still fills a gap but the UX is poor. A modernized keyboard with long-press diacritics and haptic feedback would be a straightforward improvement.

### Note on Android
LLC's own documentation now directs Android users to: **Settings → General Management → Language and Input → On-screen Keyboard → Gboard → Languages → Add Keyboard → Lakota**. The standalone Android LKT app is effectively deprecated.

## App 3: Owóksape ("Place of Wisdom")

**App Store:** https://apps.apple.com/us/app/owoksape/id1387867457
**Web:** https://owoksape.com
**Developer:** The Language Conservancy (LLC sister org)
**Price:** Free
**Platforms:** iOS, Android, Web (PWA)
**Min iOS:** 13.0–14.0
**Released:** 2018 (ongoing updates)

### Features
- Structured lesson-based curriculum (Duolingo-style)
- Four exercise modalities: listening, reading, writing, speaking
- Spaced repetition review algorithm
- Daily learning pace setting with streaks
- Gamification: points, badges, leaderboards
- Community forum: "Owóksape Village"
- Social login (Facebook, Google)
- Top Learners list (competitive)
- Adaptive review deck generation
- Regularly updated curriculum
- Cross-platform: same codebase on web and mobile (PWA wrapped in native shell)
- Covers: colors, animals, food, basic vocab, questions, grammar patterns, kinship terms

### Cultural Depth (key differentiator)
- Culturally appropriate visuals (not generic stock imagery)
- Kinship terminology built into curriculum
- Protocols for speaking with elders
- Male vs. female speech nuances in Lakota
- Verb structure depth (e.g., distinctions between hiyu, hinazin, hi, uya, i — all meaning variations of "come/go")

### Funding / Backing
- Administration for Native Americans (ANA)
- South Dakota Community Foundation
- San Manuel Band of Mission Indians
- Lush Cosmetics
- Santa Fe Natural Tobacco Company Foundation

### Strengths
- Best-positioned Lakota learning app by far — closest to Duolingo UX
- Genuine linguistic and cultural depth backed by academics (published in ACL Anthology 2020)
- Community features create social stickiness
- Cross-platform (web + iOS + Android) via PWA architecture
- Nonprofit mission = community trust

### Known Gaps (from reviews)
- Sign-up/login bugs reported (button not working for some users)
- Inconsistent update cadence
- Content thins out at advanced levels
- No AI tutor or conversational practice
- No speech recognition/pronunciation scoring
- Forum is relatively low-traffic
- No offline mode mentioned
- UI feels like a wrapped web app (because it is)

### Academic Reference
Ullrich, Thornton, Vieira, Swango & Kupiec (2020). *Owóksape — An Online Language Learning Platform for Lakota.* Proceedings of the 1st Joint Workshop on SLTU and CCURL, Marseille. https://aclanthology.org/2020.sltu-1.45/

## Competitive Gap Analysis

| Gap | Current State | Opportunity |
|-----|--------------|-------------|
| AI tutor / conversation | None | High — no app offers this |
| Pronunciation feedback | None (listen only) | High — speech recognition |
| Offline learning | NLD only | Medium — Owóksape needs it |
| Modern keyboard UX | Poor (caps default, no long-press) | Low-medium — iOS native is close |
| Cross-app integration | None — apps don't talk to each other | Medium |
| Advanced curriculum | Thin | Medium — Owóksape gaps at upper levels |
| Community/social | Low-traffic forum | Medium |
| Cultural media integration | None | High — audio stories, video |
| Web-native experience | Owóksape PWA (functional but thin) | Medium |
| Dakota dialect support | Very limited | High — separate but related language |

## Key Organizations

**Lakota Language Consortium (LLC)**
https://lakhota.org
Nonprofit — linguists, Native American leaders, volunteers. Produces NLD, LKT Keyboard, Lakota Vocab Builder, Lakota Media Player.

**The Language Conservancy (TLC)**
https://languageconservancy.org
LLC sister org. Produces Owóksape. Works with 25+ Indigenous language communities across US, Canada, Australia. Plans to replicate Owóksape model for other Indigenous languages.

## LakLangLM Positioning

LakLangLM does not compete directly with LLC/TLC. It occupies a different layer:

| | LLC/TLC Apps | LakLangLM |
|---|---|---|
| **Model** | Lesson-based curriculum | Knowledge graph exploration |
| **Metaphor** | Textbook | Living cultural archive |
| **AI** | None | Claude as tutor (conversation + quiz) |
| **Content source** | LLC linguists | Community-reviewed + archival sources |
| **Graph** | Flat word lists | Relational (words ↔ stories ↔ values ↔ people ↔ places) |
| **Story role** | Supplementary | Primary learning vehicle |
| **Access model** | Free / $9.95 | Free, open source |
| **Cultural depth** | Kinship, gender speech | Full graph: ceremonies, songs, mythology, geography |

**LakLangLM is complementary, not competitive.** It fills the "cultural preservation archive" gap that no LLC/TLC product addresses. A learner could use Owóksape for structured lessons and LakLangLM for deep cultural exploration.
