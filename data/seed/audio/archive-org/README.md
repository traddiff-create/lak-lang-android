# Internet Archive — Lakota Audio

Audio files downloaded from the Internet Archive for use in the LakLang Lakota language learning app.

## Usage in App

These audio files are cultural context material (songs/chants), not direct vocabulary instruction. Suitable for:
- Cultural modules (Cosmos Sky/Wankantanhan direction — Ceremonies/Songs)
- Background context for story screens
- Seasonal learning framework audio

## Review Status

All content enters the pipeline as `review_status: "draft"` and requires community partner approval before display to learners.

## Content Integrity

No Lakota text, translations, or descriptions were generated by AI. All metadata uses source-provided information only.
