# Lakota Language & Cultural Sources — Master Reference

A comprehensive inventory of every known digitized source for Lakota language, stories, songs, ceremonies, and cultural knowledge. Organized by ingestibility into our knowledge graph.

---

## Tier 1 — Machine-Readable, Open License (Ingest Now)

### Digital Lakota Corpus
- **URL:** https://gitlab.com/digital-lakota-corpus
- **License:** Apache 2.0
- **Format:** Text/linguistic markup, BibTeX/BibJSON
- **Content:** NLP-ready Lakota/Dakota/Nakota texts, orthographies
- **KG nodes:** Word, morpheme, orthographic variants

### Comparative Siouan Dictionary (CLLD)
- **URL:** https://csd.clld.org/ | GitHub: https://github.com/clld/csd
- **License:** Open (CLLD framework)
- **Format:** CLDF (downloadable zip), PDF
- **Content:** Cognate sets across Siouan languages, semantic categories, proto-forms
- **KG nodes:** Word (etymological), Value (semantic fields), relationships between languages

### Wikidata Lakota Entries
- **URL:** https://www.wikidata.org/wiki/Q33537
- **License:** CC0 (public domain)
- **Format:** JSON-LD, RDF via SPARQL
- **Content:** Structured entities for Lakota language, people, places
- **KG nodes:** Person, Place, Value — enrichment for existing nodes

### Wikimedia Commons — Lakota Audio
- **URL:** https://commons.wikimedia.org/wiki/Category:Lakota_language
- **License:** CC-BY 3.0+
- **Format:** Audio (WebM, MP3), video
- **Content:** ~9 files including WIKITONGUES recordings
- **KG nodes:** Audio assets for Word nodes

### Lakota Wikipedia Incubator
- **URL:** https://incubator.wikimedia.org/wiki/Wp/lkt
- **License:** CC-BY-SA 4.0
- **Format:** MediaWiki markup → JSON via API
- **Content:** Test wiki articles in Lakota

### Glosbe Lakota-English Dictionary
- **URL:** https://glosbe.com/en/lkt | API available
- **License:** Community-maintained
- **Format:** JSON/XML via REST API
- **Content:** Translation pairs, usage examples

## Tier 2 — Digitized Books, Free Access (Parse & Ingest)

### New Lakota Dictionary (Archive.org)
- **URL:** https://archive.org/details/newlakotadiction0000unse
- **Content:** 41,000+ entries, 50,000+ example sentences, 1,100 pages
- **Year:** 2008 | **Orthography:** Standardized phonemic
- **KG nodes:** Word (massive expansion), example sentences

### Dakota Grammar, Texts, and Ethnography (Riggs & Dorsey, 1893)
- **URL:** https://archive.org/details/rosettaproject_dak_vertxt-3
- **Content:** Complete grammar + extensive texts
- **KG nodes:** Word (historical), grammatical structures

### Dakota Texts (Ella Deloria, 1932)
- **URL:** https://archive.org/details/rosettaproject_dak_vertxt-2
- **Format:** PDF, EPUB, text
- **Content:** 64 traditional narratives with translations

## Tier 3 — Institutional Archives (Request Access)

### George Bushotter Texts (Smithsonian)
- **URL:** https://learninglab.si.edu/resources/view/261531
- **Content:** 3,500 pages — first Lakota texts written by a Lakota person (1887)
- **Significance:** 200+ texts on stories, customs, games, autobiography
- **KG nodes:** Story, Ceremony, Value, Person, Place

### Ella Deloria Manuscripts (American Philosophical Society)
- **URL:** https://indigenousguide.amphilsoc.org (search "Deloria, Ella Cara")
- **Content:** Unpublished stories, speeches, customs, ceremonies
- **Format:** Typewritten originals + word-for-word translations + free translations
- **KG nodes:** Story, Ceremony, Person

### James R. Walker Collection (Colorado Historical Society)
- **Content:** Correspondence, recordings, photographs, ceremonial texts
- **Informants:** Thomas Tyon (19 texts on wakan/Sun Dance), George Sword (245 pages)
- **KG nodes:** Ceremony, Story, Value, Person

### Frances Densmore Recordings (Library of Congress)
- **URL:** https://www.loc.gov/audio/?fa=language%3Adakota
- **Content:** 340+ songs on wax cylinders (1911-1914), digitized
- **Types:** Ceremonial, healing, honoring, war, hunting, sacred songs
- **KG nodes:** Song (primary source), Ceremony, Person

### Standing Rock Language Project

### Woksape Tipi Library (Oglala Lakota College, Pine Ridge)

### National Museum of the American Indian

## Tier 5 — Research & Linguistic Databases

### LakotaBERT (2025)
- **Paper:** https://arxiv.org/html/2503.18212v1
- **Training data:** 105K lines Lakota text, 700 parallel Lakota-English lines
- **Potential:** Spell-check, morphological analysis, POS tagging (95.2% accuracy)

### OLAC (Open Language Archives Community)
- **URL:** http://www.language-archives.org/language/lkt
- **Content:** Distributed archive catalog — recordings, word lists, linguistic references

### ELAR (Endangered Languages Archive)
- **URL:** https://www.elararchive.org/
- **Content:** 770+ endangered language collections, multimedia

### WALS (World Atlas of Language Structures)
- **URL:** https://wals.info/ | GitHub: https://github.com/cldf-datasets/wals
- **Content:** Typological features for Lakota phonology, grammar, lexicon

### Glottolog
- **URL:** https://glottolog.org/resource/languoid/id/siou1252
- **Content:** 460,000+ bibliographic references, language family classification

### Ethnologue
- **URL:** https://www.ethnologue.com/language/lkt/
- **Content:** Speaker demographics (<2,000 L1), geographic distribution, dialect info
